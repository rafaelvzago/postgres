# Postgres Docker

## Description

- Runs using docker-compose
- Maps the /datadir
- Don't keep the /datadir content
- mysqld.cnf stored to set the software requisites.
- Maps the 5432 port to localhost
## Requirements

- Docker
- docker-compose
- terminal :)

## How-to

```sh
	docker-compose up -d
```
